# AI 提示词与输出规范（V0）

## 系统提示词骨架
- 背景：资产类型、业务目标、受众、评估维度与权重、可用数据/约束
- 任务：生成结构化输出，字段：
  - overall_score (0-100)
  - dimension_scores: [{key, score, why}]
  - findings: 3-7 bullet
  - risks: ≤3 bullet
  - actions: [{title, why, impact, effort, owner_hint, eta}]
- 语气：简洁、可执行，避免空话；缺数据时说明并给出收集建议
- 质量守则：不得捏造；字段齐全；长度与格式校验；可溯源（引用输入数据）

## 专家角色库（示例）
- 网站体验专家：信息架构、可用性、性能、可访问性(WCAG)、内容清晰度
- 移动增长专家：激活/留存/召回、埋点与A/B、商店优化、漏斗与队列分析
- 产品策略顾问：目标-指标-举措闭环，优先级框架(ICE/RICE)，依赖与风险
- 技能评估导师：能力框架、案例与成果、学习路径、量化里程碑
- 财务健康顾问：现金流、储蓄率、债务比、风险敞口、配置建议（合规提示）

## Few-shot 模板（片段示例）
- 输入：资产类型、目标、数据摘要、维度与权重
- 输出：遵循结构化 schema；每个行动含影响与时间预估；风险为可验证陈述

## 结构化输出校验
- JSON Schema 校验字段与类型
- 必填字段检测；空值或缺失时退回降级模板
- 长度限制：actions ≤8，findings ≤7，risks ≤3
- 分数范围校验：0-100，维度分权重一致性检查

## 模型路由建议
- 默认：GPT-4/4.1
- 成本敏感/草稿：GPT-4o-mini 或同级
- 隐私或离线：自托管开源模型（准备降级提示词）

## 监控与迭代
- 采集：响应时间、令牌消耗、失败率、手工修订率
- 反馈回路：评分与修订标注 → 提示词版本化 → A/B 比较
