# 资产评级平台：需求与非功能要求

## 核心目标
构建通用、智能、可扩展的资产评级平台，覆盖评估→分析→改进闭环，输出可执行、具时效性的建议。

## 目标用户
- 专业人士：产品、研发、设计等职业优化
- 个人用户：自我提升、投资、学习
- 企业用户：业务资产与决策支持
- 教育机构：技能评估、教学设计

## 功能域
- 资产管理：创建（预设/自定义类型）、标签与集合、状态跟踪、关联/组合/继承关系
- 评估引擎：手动表单、导入(CSV/JSON/API)、自动采集、混合；向导与进度保存；模板库（预设/自定义/共享）
- AI专家分析：专家角色库、多视角协同、权重可调维度、结构化输出、个性化行动计划
- 报告与可视化：多模板（简/标/详），导出PDF/HTML/Markdown，雷达/趋势/对比/仪表盘，交互筛选
- 学习与改进：行动计划、进度提醒、成果验证；基准对比（行业/同类/历史）；学习资源与专家咨询
- 社交与协作：分享（公开/私有），团队评估与权限，评论讨论，社区模板与案例

## 架构与集成
- 原则：模块化、微服务、事件驱动、可扩展、安全
- 核心服务：网关、用户、资产、评估、AI分析、报告、通知、数据服务
- 数据层：关系型（用户/资产元数据）、文档型（评估/报告）、向量库（检索）、缓存、对象存储
- AI：提示词管理、模型路由、上下文管理、结构化输出、质量监控、持续优化；支持 GPT/Claude/开源
- 外部集成：数据采集API、认证(Auth0/OIDC)、存储(S3/OSS)、通知(SendGrid/Twilio)、支付(Stripe/PayPal)、开放API/Webhook/SDK

## 非功能需求
- 性能：API<500ms(95%)；报告<30s；页面<3s；1000+并发
- 可用性：99.9%，故障恢复<15分钟，日备份
- 安全：端到端加密、脱敏、细粒度权限、SQL注入/XSS/CSRF/暴力破解防护；GDPR/CCPA、数据保留与删除权
- 可扩展：无状态服务+LB，自动伸缩；插件式功能扩展；新数据源/服务/分析规则可插拔
- 体验：易用引导、智能默认、帮助文档；WCAG 2.1 AA，可访问性与键盘导航；响应式+移动端优化+PWA预留

## 业务流程（高阶）
1) 准备：资产选择/创建 → 评估方法 → 数据源配置  
2) 收集：采集/导入/手动输入 → 验证与清洗 → 标准化  
3) 分析：指标计算 → AI深度分析 → 基准对比  
4) 报告：内容与图表生成 → 建议与行动计划 → 导出/分享  
5) 跟进：行动跟踪 → 效果评估 → 迭代优化

## 商业与风控
- 定价：免费(限次数) / 专业 / 团队 / 企业定制（私有化+专属模型）
- 技术风险：AI稳定性、采集限制、性能、安全 → 多模型降级、多渠道采集、异步+CDN+DB优化、多层防护
- 业务风险：用户接受度、竞争、盈利、合规 → 教育与引导、差异化与生态、定价多样、法律合规设计
