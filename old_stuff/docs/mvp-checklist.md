# MVP 执行清单（1-3 个月）

## 产品
- 资产：创建/列表/详情、标签、状态（草稿/评估中/归档）
- 评估：手动向导表单，进度保存
- 报告：简洁版（总分+关键发现+3行动），雷达图数据
- 用户：注册/登录（邮箱+可选OAuth），基础分享（私密/团队内），owner/member只读

## 技术
- 架构：前后端分离或轻量单体；预留拆分接口
- 服务：网关/Auth、用户、资产、评估、报告、AI分析(雏形)
- 基础设施：Postgres/MySQL + 文档存储(JSON列或Mongo) + Redis + 对象存储(S3/OSS)；队列（异步AI/报告）
- 运维：CI/CD雏形，结构化日志，APM，错误告警；备份任务

## 数据
- 表/集合：用户、资产、标签、资产-标签、评估、评估项、模板、报告、行动计划、事件日志
- 字段：审计字段(created_at/updated_at)、tenant_id、多租户隔离、软删除
- 约束：非空/唯一/外键；JSON schema 校验（评估结果/报告）

## AI
- 模型接入：GPT系列
- 提示词：按资产类型的专家库V0；结构化输出 schema（总分、维度分、发现、风险、行动列表）
- 质量：长度与必填校验；超时降级为简版模板

## 安全
- HTTPS、JWT/OAuth2、RBAC
- 输入校验、CSRF/XSS/SQL注入防护、速率限制
- 审计日志、异常告警、每日全量备份（保留≥7天）
