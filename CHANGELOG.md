# Changelog - Asset Rating Platform

All notable changes to this project are documented below.

## [1.0.0] - 2025-12-24

### Infrastructure & Monorepo
- **Turbo Setup**: Initialized Turborepo with `api` (NestJS) and `web` (Next.js) apps.
- **PNPM Transition**: Migrated from npm to pnpm for efficient workspace management.
- **Database**: 
  - Integrated Prisma ORM (v5.22.0).
  - Configured PostgreSQL via `docker-compose.yml`.
  - Defined comprehensive `schema.prisma` with 7 core entities.

### Backend (apps/api)
- **Core Modules**: Implemented `Tenants`, `Users`, `Assets`, and `Tags` with full CRUD capabilities.
- **Evaluation Engine**: 
  - Created `Templates` module for flexible scoring dimensions.
  - Created `Evaluations` module to track scoring sessions and progress.
- **AI Analysis**: 
  - Implemented `AiService` for automated asset analysis (mock logic ready for LLM integration).
  - Added `AiController` to handle analysis requests.
- **Reporting**: 
  - Implemented `Reports` module for summary generation and detailed evaluation reviews.
- **Architecture**: Established a global `PrismaModule` and standardized dependency injection.

### Frontend (apps/web)
- **Design System**: 
  - Configured Tailwind CSS 3 with a customized Shadcn-compatible theme.
  - Implemented a premium glassmorphic UI with `lucide-react` icons.
- **Layout & Navigation**: 
  - Created a responsive `WebLayout` with a persistent `Sidebar`.
  - Dynamic navigation highlighting based on active path.
- **Dashboard**: Implemented a modern overview with stats, recent activity, and quick actions.
- **Management Pages**: 
  - Dedicated `Assets` page with a filterable data table.
  - `Evaluations` list with real-time progress bars and score displays.
  - `Templates` library with card-based grid view.
- **Evaluation Wizard**: Built a multi-step stepper component for conducting new evaluations.

### Shared Packages (packages/api)
- **Type Safety**: Defined shared DTOs (Data Transfer Objects) and Entities for all modules.
- **Sync**: Unified API surface ensures frontend and backend stay perfectly in sync.
